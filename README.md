# 信道编码课程复习笔记

本仓库包含信道编码（Channel Coding）课程的系统复习笔记，涵盖了从信息论基础到现代编码技术的核心内容。

## 📚 笔记内容

本系列笔记按照信道编码的理论发展脉络组织，共包含以下内容：

### 基础理论
- **编码理论的信息论基础** (`s2_channel_coding_theorem_signed.pdf`)
  - Berlekamp-Massey算法
  - 渐近均分性定理(AEP)
  - 信道编码定理及其逆定理
  - 联合典型序列与费诺不等式

### 线性分组码
- **线性分组码及代数译码** (`s3_linear_block_code_signed.pdf`)
  - 生成矩阵与校验矩阵
  - 标准阵与校正子译码
  - 汉明码与完全码
  - 循环码及其编解码
  - BCH码与RS码

### 卷积码
- **卷积码** (`s4_convolutional_code_signed.pdf`)
  - 多项式与矩阵表示
  - 网格图表示
  - Viterbi译码算法
  - BCJR译码算法

### 现代编码技术
- **Turbo码** (`s5_turbo_code_signed.pdf`)
  - 并行级联卷积码结构
  - MAP译码算法
  - 迭代译码与外信息传递

- **低密度奇偶校验码(LDPC)** (`s6_LDPC_code_signed.pdf`)
  - 稀疏校验矩阵构造
  - Gallager与Mackay构造方法
  - 和-积译码算法（置信度传播）
  - 树图与因子图表示

- **极化码(Polar Code)** (`s7_polar_code_signed.pdf`)
  - 信道极化现象
  - 编码矩阵构造
  - 信道可靠性估计
  - 逐次消去(SC)译码算法
